# pycharm configuration





### pycharm软件Edit configurations配置

1、Name（自己取的） 

2、Script是项目文件的路径 

**3、Python interpreter（解释器）是事先安装好的Python解释器的路径** 

4、Working directory是你项目的路径（2是其子文件）。然后依次点击Apply，点击OK即可
